## Word Cloud

Simple Word Cloud Experiment

## Version

0.0.1

## Dependencies

- [node.js and npm](https://gist.github.com/isaacs/579814)
- [bower](http://bower.io/)

## Running

    git clone git@github.com:francesko/wordcloud.git
    cd wordcloud
    npm install
    bower install
    grunt serve

## Tests

    grunt test
    grunt serve:test
